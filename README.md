# IKOREX — Intelligent Automation & Operations

IKOREX is a premium, high-velocity enterprise process automation portal engineered for precision overhead reduction, compliant compliance architectures, and autonomous system workflows. Designed with an ultra-premium, dark/light cybernetic Apple-like aesthetic.

## Features

- **Modern Adaptive UX**: Seamless transitions and dynamic staggered animation layers built on vanilla HTML5, CSS3, and JavaScript.
- **Cyber-Blueprint Grid & Glow Loader**: High-tech telemetry loading sequence matching the homepage grid coordinates.
- **Responsive Layout**: Precision-tuned layout breakpoints adapted to all modern device viewports (desktop down to 320px mobile screens).
- **Dual Theme Support**: Beautiful light and dark themes with persistent storage and instant toggle adaptation.
- **Interactive Metrics & Spotlight Tracking**: Custom cursor hover spotlight effects on interactive cards.

## Technology Stack

- **Frontend**: Vanilla HTML5, CSS3 Variables, modern typography (Syne, Plus Jakarta Sans, Share Tech Mono).
- **Logic**: Vanilla ES6 JavaScript for dynamic transitions and state tracking.
- **Animations**: CSS custom timing curves (`cubic-bezier`).
